void ugit_version ( void )

Imprime la versión actual del sistema de control de versiones uGit.

Muestra la versión actual del sistema uGit.

La función ugit_version muestra en la consola la versión actual del software uGit. Esto es útil para verificar qué versión del sistema se está utilizando y puede ser especialmente relevante durante el desarrollo y mantenimiento del software.

Returns
No devuelve ningún valor. La función imprime la versión en la salida estándar.

Al llamar a esta función, se imprimirá un mensaje que incluye el número de versión del sistema de control de versiones uGit. La salida es un simple mensaje de texto con el formato ugit version X.X.X, donde X.X.X representa la versión actual del software.
