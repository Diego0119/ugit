Detailed Description

Permite cambiar el prompt de la shell basado en la rama actual.

Esta función actualiza el prompt de la shell para reflejar la rama en la que se encuentra el usuario. El prompt se ajusta para mostrar el nombre de la rama actual como parte del texto del prompt.

Cambia el prompt de la shell para reflejar la rama actual.

Controla las operaciones relacionadas con ramas.

La función ugit_branch_control actualiza el prompt de la shell (actual_prompt) para incluir el nombre de la rama actual. Si el comando prompt es "main", el prompt se establece con el nombre de la rama principal "main". Para otros nombres de ramas, se extrae el nombre de la rama del comando prompt.

Parameters
promptCadena de caracteres que contiene el comando actual. Si el comando es "main", el prompt se configura para mostrar "main". De lo contrario, se extrae el nombre de la rama del comando.
actual_promptCadena de caracteres donde se almacenará el nuevo texto del prompt de la shell.
void ugit_branch_control(char *prompt, char *actual_prompt)
{
char *branch_name;
if (strcmp(prompt, "main") == 0)
{
branch_name = "main";
}
else
{
branch_name = prompt + 14;
}
snprintf(actual_prompt, MAX_INPUT_SIZE, "ugit(%s) ", branch_name);
}
