\doxysubsection{Detailed Description}
Permite visualizar el historial de commits realizados en una rama específica. 

Esta función muestra el historial de commits realizados en la rama actual del sistema de control de versiones {\ttfamily u\+Git}. Imprime detalles de cada commit, incluyendo el autor, el hash del commit y la descripción asociada.

Muestra el historial de commits de una rama. 

Muestra el historial de commits.

La función {\ttfamily ugit\+\_\+log} imprime el historial de commits de la rama especificada. Muestra información sobre cada commit, incluyendo el autor (nombre del usuario), el hash del commit y la descripción del commit.

La función realiza las siguientes operaciones\+:
\begin{DoxyItemize}
\item Obtiene el número total de commits en la rama especificada utilizando el contador de commits en la estructura de datos {\ttfamily branches}.
\item Imprime una línea separadora para mejorar la legibilidad de la salida.
\item Si hay commits disponibles ({\ttfamily commit\+\_\+count \texorpdfstring{$>$}{>} 0}), itera a través de cada commit e imprime\+:
\begin{DoxyItemize}
\item El nombre del autor del commit ({\ttfamily user\+\_\+name}).
\item El hash del commit.
\item La descripción del commit.
\item Una línea separadora después de cada commit para facilitar la lectura.
\end{DoxyItemize}
\item Si no hay commits disponibles, imprime un mensaje indicando que no hay commits.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Solo se mostraran los logs de la rama en la cual se haga el {\ttfamily ugit log}. 
\end{DoxyNote}
