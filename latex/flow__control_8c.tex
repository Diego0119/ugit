\doxysubsection{Detailed Description}
Implementación de la función de control de flujo para la interacción del usuario con u\+Git. 

Este archivo contiene la implementación de la función {\ttfamily flow\+\_\+control}, que permite al usuario interactuar con el sistema de control de versiones u\+Git a través de una interfaz de línea de comandos. La función lee comandos del usuario, los interpreta y ejecuta las funciones correspondientes según el comando ingresado. También maneja la inicialización del repositorio, la creación de ramas, la adición de archivos, y la realización de commits, entre otros.
