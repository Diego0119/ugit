\doxysubsection{Detailed Description}
Permite crear nuevas ramas en el sistema de control de versiones. 

Esta función se encarga de crear una nueva rama en el sistema, utilizando el nombre proporcionado en el comando {\ttfamily prompt}. Si el nombre de la rama es vacío, se muestra un mensaje de error. La nueva rama se inicializa con un contador de commits en cero.

Crea una nueva rama en el sistema de control de versiones. 

Controla la creación de nuevas ramas.

La función {\ttfamily ugit\+\_\+branch} agrega una nueva rama a la estructura de datos {\ttfamily branches}. La función verifica si el nombre de la rama es "{}main"{}, y si no, toma el nombre de la rama del comando {\ttfamily prompt}. Si el nombre de la rama es vacío, se muestra un mensaje de error y la función termina.

\hline
{\em branches} & Estructura de datos que contiene todas las ramas y commits. La nueva rama se agrega a esta estructura.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
