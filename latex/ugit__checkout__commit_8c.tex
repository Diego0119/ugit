\doxysubsection{Detailed Description}
Permite realizar el checkout de un commit específico en una rama. 

Esta función permite al usuario cambiar al commit especificado en una rama determinada. Se busca el commit en la rama basándose en el hash del commit proporcionado como cadena de caracteres y se compara con los commits almacenados en la rama.

La función realiza las siguientes operaciones\+:
\begin{DoxyItemize}
\item Convierte el hash del commit de cadena a número entero usando {\ttfamily strtol}.
\item Itera a través de los commits en la rama correspondiente y compara el hash convertido con los hashes almacenados.
\item Si encuentra el commit correspondiente, imprime un mensaje indicando el cambio de commit y retorna 1.
\item Si no se encuentra el commit, muestra un mensaje de error y retorna 0.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
La conversión de la cadena {\ttfamily commit\+\_\+hash} a número entero puede fallar si la entrada no es válida. 
\end{DoxyNote}
